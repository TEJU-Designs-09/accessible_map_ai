# 🚀 Accessible Map AI

Accessible Map AI is an AI-powered smart mobility web application designed to provide **safe, inclusive, and accessible navigation** for all users — especially wheelchair users, elderly individuals, and visually impaired travelers.

It combines **intelligent routing, accessibility awareness, safety scoring, and real-time simulation features** into one unified platform.

---

# 🌟 Key Features

## 🧭 Smart Accessible Routing

• Step-free path optimization
• Avoid stairs & unsafe roads
• Wheelchair-friendly navigation
• Real road routing using OSMnx

## ⚠️ AI Safety Intelligence

• Accident-prone zone detection
• Risk-based route scoring
• Smart alerts system
• Explainable AI insights

## 🗺️ Interactive Map Experience

• Real route visualization
• Crowd heatmap overlay
• Smart markers & directions

## 🔊 Smart Navigation Features

• Voice guidance system

• Live demo navigation mode

## 📊 Intelligent Dashboard

• AI safety score

# ⚙️ How to Run

## 1️⃣ Clone Repo

```bash
git clone https://github.com/YOUR_USERNAME/accessible_map_ai.git
cd accessible_map_ai
```

## 2️⃣ Run Backend

```bash
cd backend
python -m uvicorn app.main:app --reload
```

## 3️⃣ Run Frontend

```bash
cd frontend
streamlit run app.py
```
